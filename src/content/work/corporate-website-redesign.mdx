# Редизайн корпоративного сайта

## О проекте

Полный редизайн корпоративного сайта для международной компании в сфере консалтинга. Проект включал обновление визуальной идентичности, улучшение пользовательского опыта, оптимизацию производительности и реализацию многоязычной поддержки для 12 различных локаций.

## Технологии

- **Frontend**: TypeScript, Next.js 13, Tailwind CSS
- **CMS**: Contentful
- **Локализация**: i18next, next-intl
- **Performance**: Next.js App Router, RSC
- **Аналитика**: Google Analytics 4, Tag Manager
- **CI/CD**: GitHub Actions, Vercel

## Ключевые аспекты проекта

### Комплексное исследование пользователей

На начальном этапе было проведено:
- Интервью с ключевыми стейкхолдерами
- Анализ поведения пользователей на существующем сайте
- Исследование конкурентов
- Разработка пользовательских персон
- Создание карт пользовательских путей

### Улучшение производительности

Значительно улучшены показатели производительности:

| Показатель | До | После |
|------------|-----|-------|
| Lighthouse Performance | 64 | 97 |
| First Contentful Paint | 2.4s | 0.6s |
| Largest Contentful Paint | 4.8s | 1.2s |
| Cumulative Layout Shift | 0.28 | 0.02 |

### Многоязычная поддержка

Реализована полноценная мультиязычная поддержка с:
- Автоматическим определением локали пользователя
- Переключением между языками без перезагрузки страницы
- Оптимизированной структурой URL для SEO
- Автоматическим переводом динамического контента через интеграцию с API Contentful

### Оптимизация для SEO

Применены современные практики SEO:
- Структурированные данные (schema.org)
- Оптимизированная структура заголовков
- Мета-теги с учетом специфики каждой локали
- Динамическая генерация sitemap.xml
- Интеграция с Google Search Console

## Результаты

После запуска нового сайта:
- Увеличение органического трафика на 47%
- Снижение показателя отказов на 38%
- Увеличение времени на сайте на 2 минуты 15 секунд
- Рост конверсии в заполнение формы обратной связи на 62%
- Повышение удовлетворенности пользователей по результатам опроса на 45%
