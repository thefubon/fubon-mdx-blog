# Мобильное приложение для банкинга

## О проекте

Разработка нативного мобильного приложения для крупного регионального банка, позволяющего клиентам управлять финансами, совершать переводы, оплачивать услуги и получать финансовую аналитику. Особое внимание было уделено безопасности и удобству использования.

## Технологии

- **Mobile**: React Native, TypeScript
- **State Management**: Redux Toolkit
- **Безопасность**: Biometric Auth, End-to-end шифрование
- **Backend**: Java Spring Boot
- **API**: REST, GraphQL
- **Analytics**: Firebase Analytics, Crashlytics

## Ключевые особенности

### Безопасность без компромиссов

Приложение обеспечивает высокий уровень безопасности благодаря:
- Биометрической аутентификации (Face ID / Touch ID)
- Многофакторной аутентификации
- Шифрованию данных при хранении и передаче
- Автоматическому завершению сессий
- Защите от скриншотов конфиденциальной информации

### Умная персонализация

Приложение использует алгоритмы машинного обучения для:
- Предложения релевантных финансовых продуктов
- Анализа расходов и предложения стратегий экономии
- Прогнозирования будущих трат на основе истории операций

### Улучшенный UX

Было проведено более 50 сессий с фокус-группами для разработки интуитивно понятного интерфейса:
- Доступная навигация с минимальным количеством тапов для достижения любой цели
- Персонализированная домашняя страница
- Удобные виджеты быстрого доступа
- Темная тема и настраиваемый дизайн

## Результаты проекта

После запуска приложения в производство:
- 125,000+ скачиваний в первый месяц
- Рейтинг 4.8/5 в App Store и 4.7/5 в Google Play
- Более 68% активных пользователей ежедневно
- Рост транзакций через мобильное приложение на 87%
- Снижение нагрузки на колл-центр на 34%
