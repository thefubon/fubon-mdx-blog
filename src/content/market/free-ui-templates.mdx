# Free UI Icons Pack

Коллекция из более чем 100 бесплатных иконок для ваших веб-проектов, доступных в форматах SVG и PNG.

## Что входит в набор

- 100+ иконок в минималистичном стиле
- Форматы SVG и PNG
- 3 размера (16px, 24px, 32px)
- Две цветовые темы (светлая и темная)
- Полностью настраиваемые SVG
- Оптимизированные файлы

## Категории иконок

- Интерфейс пользователя (стрелки, кнопки, меню)
- Социальные сети
- Электронная коммерция
- Мультимедиа
- Файлы и документы
- Погода
- И многое другое!

## Как использовать

После скачивания вы получите ZIP-архив со следующей структурой:

```
free-ui-icons/
├── svg/
│   ├── light/
│   └── dark/
├── png/
│   ├── 16px/
│   ├── 24px/
│   └── 32px/
└── license.txt
```

Для использования SVG-иконок вы можете просто скопировать код SVG или использовать изображение как обычный файл:

```html
<!-- Вариант 1: Вставка SVG кода -->
<svg width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">
  <!-- код иконки -->
</svg>

<!-- Вариант 2: Использование как изображение -->
<img src="path/to/icon.svg" alt="Icon" width="24" height="24">
```
