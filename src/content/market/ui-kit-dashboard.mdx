# UI Kit Dashboard

Полный набор компонентов для создания современной админ-панели с использованием React и Tailwind CSS.

## Что входит в набор

- Более 50 готовых компонентов
- 10 готовых страниц (дашборд, аналитика, настройки и др.)
- Темная и светлая темы
- Полностью адаптивный дизайн
- Документация по использованию

## Технологии

- React 18+
- Tailwind CSS 3.0
- TypeScript
- Next.js совместимость
- Vite совместимость

## Преимущества

- Готовые компоненты для быстрого старта
- Современный дизайн
- Высокая производительность
- Легкая кастомизация
- Регулярные обновления

## Как использовать

После покупки вы получите доступ к архиву с исходным кодом. Распакуйте архив и следуйте инструкциям в README.md файле для установки и настройки проекта.

```bash
# Установка зависимостей
npm install

# Запуск проекта
npm run dev
```
