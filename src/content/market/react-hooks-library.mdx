# React Hooks Library

Библиотека из 30+ готовых React хуков для ускорения разработки веб-приложений.

## Что входит в библиотеку

- Хуки для работы с формами (`useForm`, `useField`, `useValidation`)
- Хуки для работы с API (`useFetch`, `useQuery`, `useMutation`)
- Хуки для работы с UI (`useModal`, `useToast`, `useMediaQuery`)
- Хуки для работы с состоянием (`useLocalStorage`, `useSessionStorage`, `useReducerState`)
- Хуки для оптимизации производительности (`useDebounce`, `useThrottle`, `useMemoCompare`)

## Преимущества

- Полностью типизированы с использованием TypeScript
- Минимальные зависимости
- Оптимизированы для производительности
- Подробная документация и примеры использования
- Совместимы с React 16.8+ и React Native

## Как использовать

После покупки вы получите доступ к архиву с исходным кодом библиотеки. Вы можете использовать хуки непосредственно в своем проекте или установить как npm-пакет.

```jsx
import { useForm, useLocalStorage } from 'react-hooks-library';

function MyComponent() {
  const { values, handleChange, handleSubmit } = useForm({
    initialValues: { name: '', email: '' },
    onSubmit: (values) => console.log(values),
  });
  
  const [theme, setTheme] = useLocalStorage('theme', 'light');
  
  return (
    // ваш компонент
  );
}
```
