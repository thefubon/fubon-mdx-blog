# Next.js Blog Template

Полностью готовый к использованию шаблон блога на Next.js с поддержкой MDX, темной темы и SEO-оптимизацией.

## Особенности шаблона

- Построен на Next.js App Router
- Поддержка MDX для написания статей
- Темная и светлая темы
- Полная SEO-оптимизация
- Адаптивный дизайн
- Интеграция с аналитикой
- Комментарии (опционально)
- Поиск по блогу
- Категории и теги
- Пагинация

## Технический стек

- Next.js 13+
- TypeScript
- Tailwind CSS
- MDX
- next-mdx-remote
- next-themes
- next-seo

## Как использовать

После покупки вы получите доступ к репозиторию с исходным кодом. Клонируйте репозиторий и следуйте инструкциям в README.md.

```bash
# Клонирование репозитория
git clone [ваш-доступ-к-репозиторию]

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

## Настройка

Шаблон легко настраивается через конфигурационные файлы:

1. `site-config.js` - основные настройки сайта
2. `theme-config.js` - настройки темы и цветов
3. `.env.local` - переменные окружения

## Создание контента

Добавление новых статей происходит через создание MDX файлов в директории `content/posts/`. Каждый файл должен содержать frontmatter с метаданными статьи.
