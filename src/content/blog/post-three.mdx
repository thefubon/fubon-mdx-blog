## Что такое MDX?

MDX позволяет использовать JSX в ваших Markdown документах. Вы можете импортировать компоненты, такие как интерактивные диаграммы или оповещения, и встраивать их в ваш контент.

```jsx
function Button() {
  return (
    <button onClick={() => alert('Нажата кнопка!')}>
      Нажми меня
    </button>
  )
}

<Button />
