# Все, что нужно знать о Next.js 13 App Router

Next.js 13 представил революционный подход к маршрутизации в React-приложениях - App Router. Этот новый подход заменяет прежний Pages Router и предоставляет более интуитивный способ организации кода.

## Основные преимущества App Router

1. **Вложенные маршруты** - структура папок отражает структуру URL, что делает навигацию более интуитивной
2. **Серверные компоненты** - по умолчанию компоненты рендерятся на сервере
3. **Параллельные маршруты** - возможность одновременно отображать несколько страниц
4. **Интерсептирующие маршруты** - перехват и модификация навигации
5. **Улучшенная производительность** - оптимизации на уровне маршрутизации

## Структура проекта с App Router

```jsx
app/
  layout.js      // Корневой макет
  page.js        // Главная страница (/)
  about/
    page.js      // Страница /about
  blog/
    [slug]/
      page.js    // Динамическая страница блога
  (dashboard)/   // Группа маршрутов
    layout.js    // Макет для группы
    page.js      // Страница /dashboard
```

## Серверные и клиентские компоненты

Одно из ключевых нововведений - все компоненты по умолчанию являются серверными. Это позволяет:

- Уменьшить размер JavaScript на клиенте
- Улучшить первоначальную загрузку страницы
- Обеспечить лучшую SEO оптимизацию

Для создания клиентского компонента используется директива `'use client'`:

```jsx
'use client'

import { useState } from 'react'

export default function Counter() {
  const [count, setCount] = useState(0)
  
  return (
    <div>
      <p>Счетчик: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Увеличить
      </button>
    </div>
  )
}
```

## Загрузка данных

App Router предоставляет нативные функции для асинхронного получения данных прямо в компонентах:

```jsx
async function getDashboardData() {
  const res = await fetch('https://api.example.com/dashboard')
  return res.json()
}

export default async function Dashboard() {
  const data = await getDashboardData()
  
  return (
    <main>
      <h1>Панель управления</h1>
      {data.items.map(item => (
        <div key={item.id}>{item.name}</div>
      ))}
    </main>
  )
}
```
